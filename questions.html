[1] É a regra do(a) __________ que nos diz como calcular a derivada de expressões da forma xn (em outras palavras, expressões com x elevado à qualquer potência):

A) Regra da potência

B) Regra da multiplicação por constante

C) Regra do produto

D) Regra do Cociente

Resposta correta: letra C

[2] Dada a função f(x) = 2. Qual a derivada dessa função?

A) f’(x) = 2x²

B) f’(x) = x²

C) f’(x) = 0

D) f’(x) = 4x

Resposta correta: letra C


[3] A derivada de uma parábola é:

A) Uma reta paralela ao eixo x

B) Uma reta crescente

C) Uma reta decrescente

D) Uma reta

Resposta correta: letra D


[4] A taxa de variação de uma curva quando seu intervalo tende a zero, é conhecida como:

A) Soma de quadrados

B) Regra do cociente

C) Derivada

D) Limite

Resposta correta: letra C


[5] Qual a derivada de f(x) = 3x⁵ – 2x³ + 5 – 3x?

A) 3x⁴ - 2x² - 2

B) x² – 3

C) 15x⁴ – 6x² - 3

D) -2x³ + 5

Resposta correta: letra C


[6] Ao derivarmos f(x) = x² no ponto 2:

A) f’(x) = 2

B) f’(2) = 2 * 2

C) f’(2) = 4

D) --

Resposta correta: letra C


[7] Qual a deriva de f(x) = x³ + 1000?

A) f’(x) = 3x²

B) f’(x) = 3x² - 1000

C) f’(x) = x²

D) f’(x) = 3x³

Resposta correta: letra A


[8] Com base na função f(x) = x³ + x² + x + 1. Ao calcularmos sua derivada temos:

A) f’(x) = 3 + 2x + 1

B) f’(x) = 3x² + 2x + 1x²

C) f’(x) = 3x² + 2x + 1

D) f’(x) = x³ + 2x + 2

Resposta correta: letra C


[9] Das regras abaixo qual NÃO pertence a regra da derivada?

A) Regra da Potência

B) Regra da multiplicação por constante

C) Regra da Soma

D) Regra do Polinomio

Resposta correta: letra D


[10] Qual o valor de y = x⁵?

A) y⁴

B) xy⁵⁻¹

C) x⁴

D) x⁵⁻¹

Resposta correta: letra C


[11] O conceito de _________ estuda a variação das funções, como uma dada função varia na medida em que variamos o seu valor de x.

Com isso podemos saber se a função cresce e qual a taxa de crescimento dela. Um uso muito comum serve para identificar pontos máximos e mínimos de uma função.

A) Derivada

B) Integral

C) Limite

D) Correlação

Resposta correta: letra A


[12] Ao calcularmos a derivada de f(x) = 6x³ − 9x + 4. Obtemos:

A) f′(x) = 18x² − 9

B) f′(x)=18x² − 3x + 2

C) f′(x)=3x² − 9

D) f′(x)=18x² − 3

Resposta correta: letra A


[13] Defina a derivada da função: f(x)=2x⁴ − 10x2 + 13x:

A) 4x² − 20x + 13

B) 8x³ − 20x + 13

C) 8x³ − 100x

D) 8x³ − 20x + 3x

Resposta correta: letra B


[14] Qual a derivada da função f(x) = x²

A) f'(x) = 2x

B) f'(x) = 4x

C) f'(x) = 2x2

D) f'(x) = x4

Resposta correta: letra A


[15] Dada a função f(x) = 20, podemos dizer que sua derivada é:

A) É igual a 0

B) Uma constante

C) f'(x) = 20x

D) Alternativas A e B

Resposta correta: letra D


[16] Com base na função f'(x) = 5x³ + 2x, a derivada correspondente a ela é:

A) f'(x) = 5x² + 2

B) f'(x) = 15x² + 2

C) f'(x) = 3x5 - 2x

D) f'(x) = 15x² + 2x

Resposta correta: letra B


[17] A derivada de uma função é um(a) __________.

A) Soma

B) Fração

C) Função

D) Quociente

Resposta correta: letra C


[18] Qual destes filósofos/matemáticos descobriu a Regra do Produto?

A) Gottfried Wilhelm Leibniz

B) Isaac Newton

C) Augustin Louis Cauchy

D) Karl Weierstrass

Resposta correta: letra A


[19] As derivadas quando existem determinam:

A) A oscilação de valores a um limite

B) A o comprimento dos pontos (A, B) em uma reta

C) A inclinação da reta tangente a uma função f(x)

D) A soma dos componentes dentro da função

Resposta correta: letra C


[20] Qual a derivada de e(x)(b+1)?

A) x (b-1)

B) (b-1)x

C) x(1-b)

D) (b+1)xb

Resposta correta: letra D
